# UI Checklist: スペース詳細画面

このファイルはスペース詳細画面 (`/spaces/:id`) における表示レベル（UI）の要件を管理するチェックリストです。修正を行った際は、対象の要件を満たしているかを検証してください。

## 削除確認
- [x] **スペース削除確認の画面内ダイアログ化**:
  右上の「削除」を押した際、ブラウザ標準の確認ダイアログではなく、アプリ内の確認ダイアログが表示され、確認後に削除処理が進むこと。

## 成功通知
- [x] **ジョブ実行通知の toast 化**:
  所属ジョブ一覧の「実行」を押した際、ブラウザ標準のアラートではなく、画面内 toast で実行トリガー完了が通知されること。

## 一覧表記
- [x] **ジョブ種別表記の英字統一**:
  所属ジョブ一覧の `タイプ` 列では `Cron / DAG / OneShot` の英字表記のみを表示し、日本語補足を出さないこと。
- [x] **スケジュール未設定時のハイフン表示**:
  所属ジョブ一覧の `スケジュール` 列では未設定時に `-` を表示し、`なし` や `未設定` の表記揺れを出さないこと。
