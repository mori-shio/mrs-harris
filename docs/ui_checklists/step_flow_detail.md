# ステップフロー詳細 UI チェックリスト

- [x] **概要表示**:
  ステップフロー名、説明、最新バージョン、有効状態が確認できること。
- [x] **パンくずへの所属スペース表示**:
  ステップフロー詳細画面のパンくずリストでは、一覧の `すべて` タブから開いた場合でも対象ステップフローの所属スペースを表示し、未所属の場合は `未分類` を表示すること。
- [x] **Group/Step 表示**:
  各 Group の実行条件と、その中に含まれる Step の参照ジョブが確認できること。
- [x] **実行導線**:
  `実行` ボタンからステップフロー実行履歴を作成し、実行詳細画面へ遷移できること。
