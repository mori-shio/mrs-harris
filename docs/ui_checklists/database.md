# UI Checklist: データベース管理画面

このファイルはデータベースブラウザ画面などにおける表示レベル（UI）の要件を管理するチェックリストです。

## テーブル・データ表示
 - **横スクロールのサポート**:
  表示項目（カラム）が多く右に長い場合、画面外にはみ出して見えなくならないよう、テーブル部分に横向きのスクロールバーが表示されること。
 - **カード幅の最適化**:
  テーブル一覧やデータを表示するカードの横幅が、画面の表示領域内に収まる（はみ出さない）ように調整されていること。
 - **縦幅の動的調整と縦スクロールの排除**:
  テーブル一覧カードの縦幅が、中のコンテンツ（テーブルの数）に合わせて適切に伸縮し、カード内部に無駄な縦スクロールバーが出現しないこと。
