# ステップフロー一覧 UI チェックリスト

- [x] **サイドバー導線**:
  サイドバーの `ジョブ管理` 配下に `ステップフロー` が表示され、`/step-flows` に遷移できること。
- [x] **一覧表示**:
  ステップフロー一覧では名前、説明、タグ、有効状態、詳細導線が確認できること。
- [x] **新規作成導線**:
  右上の `+ 新規作成` から `/step-flows/new` に遷移できること。
- [x] **コピーして作成導線**:
  右上の `コピーして作成` ボタンを押すと既存ステップフローを前方一致検索できるモーダルが開き、選択すると `/step-flows/new?copy_from=<step_flow_name>` へ遷移できること。
- [x] **ジョブ一覧相当の一覧構成**:
  ステップフロー一覧はジョブ一覧と同様に、スペースタブ、検索用カード、一覧カードの順で表示されること。
- [x] **スペースタブ**:
  `すべて`、登録済みスペース、`未分類` のタブで一覧を絞り込めること。
- [x] **スペース選択時のパンくず表示**:
  スペースタブを `すべて` 以外に切り替えた場合、パンくず末尾に選択中スペースが表示され、登録済みスペースは `folder`、未分類は `help-circle` アイコンが表示されること。
- [x] **検索カード**:
  名前/説明のキーワード検索と有効/無効ステータスの絞り込みができること。
- [x] **一覧カード**:
  一覧カードにはステップフロー名、説明、ステータス、タグ、詳細導線がこの順で表示され、スペースカラムは表示されないこと。
- [x] **タグ表示**:
  ステップフロー一覧のタグはジョブ一覧と同じチップ表示、折りたたみ、省略、tooltip 表示で扱われること。
